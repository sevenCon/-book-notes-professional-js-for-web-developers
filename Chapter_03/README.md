### 3.4 数据类型
js 中浮点数的相加 遵循 IEEE754 的标准的通病，es 并非就此一家，其他使用相同数值的语言也都有。
比如： 0.15+0.25 = 0.4  这个的相加就无此问题。
```
<!-- 浮点数的相加相等 -->
    var nmb_a = 0.15;
    var nmb_b = 0.25;
    var all = 0.4;
    if(nmb_a + nmb_b == all){
        console.log("浮点数的相加 "+nmb_a+" + "+ nmb_b + " == "+all+"?", "相等！");
    }else{
        console.log("浮点数的相加 "+nmb_a+" + "+ nmb_b + " == "+all+"?", "不相等！");
    }
    <!-- 就是这么玄乎 -->
    // ==>  0.1+0.2 = 0.30000000000000004 
    // ==>  0.15+0.25 = 0.4   
```
